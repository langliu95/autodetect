API Summary

Observable variable model

For stand-alone models without torch.nn.Module:

autodetect.information(theta, obs, loglike)

Compute score function and information matrix.

autodetect.autograd_test(theta, obs, loglike)

Compute autograd-test statistics.

For models inherited from torch.nn.Module:

autodetect.AutogradTest(pretrained_model, ...)

A class for autograd-test for models inherited from torch.nn.Module.

Latent variable model

autodetect.AutogradHmm(n_states[, init])

A class for autograd-test in hidden Markov model.

Time series model

autodetect.AutogradHmm(n_states[, init])

A class for autograd-test in hidden Markov model.

Text topic model

autodetect.AutogradTopic(num_cat, num_hid[, ...])

A class for autograd-test in text topic (Brown) model.

Data

For simulation studies:

autodetect.data.Generator(size, dim, tau)

A class for sampling synthetic data with change in parameters.

Utils

For sampling Bessel processes:

autodetect.utils.sample_square_Bessel(df, ...)

Generate a sample path from square Bessel process.

autodetect.utils.sample_max_square_Bessel(df, ...)

Generate a sample from the maximum of square Bessel process.

autodetect.utils.quantile_max_square_Bessel(q, ...)

Generate quantiles of the maximum of square Bessel process.
